AI 视听语言研究术语字典

这个字典把综述中高频出现的英文术语和影视学概念放在一起解释。每个词条都说明它是什么意思、为什么重要、在研究中怎么用，以及适合继续学习的资料。

shot length / 镜头长度

也可译为：镜头时长、单镜头持续时间

定义：一个镜头从开始到结束持续的时间，通常以秒为单位。它是电影形式量化分析中最基础的指标之一。

为什么重要：镜头长度直接关系到剪辑节奏、观看速度、紧张感和叙事密度。动作片、艺术电影、纪录片、电视剧和短视频的镜头长度分布往往差异很大。

怎么使用：研究者可以统计某部作品或某位导演的镜头长度分布，比较不同年代、类型或导演风格的剪辑节奏。

示例问题：某导演后期作品的镜头是否越来越短？短视频爆款是否具有更高的剪辑频率？

ASL / Average Shot Length / 平均镜头长度

常见缩写：ASL

定义：一部影片或一个片段中所有镜头长度的平均值。通常计算方式是总时长除以镜头数量。

为什么重要：ASL 可以快速描述一部作品的整体剪辑速度。ASL 越短，通常意味着剪辑越密集；ASL 越长，通常意味着节奏更舒缓或长镜头比例更高。

局限：ASL 是平均值，容易掩盖内部差异。两部影片 ASL 相同，也可能一个镜头分布稳定，另一个由大量短镜头和少数长镜头组成。因此最好结合中位数、分布图和段落分析。

写综述时可以把 ASL 作为“传统统计特征如何进入电影风格研究”的典型例子。

shot scale / 景别

常见类别：远景、全景、中景、近景、特写、大特写

定义：景别描述主体在画面中所占比例和观看距离。它决定观众与人物、空间和动作之间的感知距离。

为什么重要：景别是视听语言中最核心的镜头元素之一。特写常用于强调情绪和细节，远景常用于展示环境、空间关系和人物处境。

AI 任务：shot scale recognition，即自动识别镜头属于远景、中景、近景或特写等类别。

shot boundary / 镜头边界

相关任务：shot boundary detection，镜头边界检测

定义：镜头边界是一个镜头结束、下一个镜头开始的位置。它可能是硬切，也可能是淡入淡出、叠化或其他转场。

为什么重要：没有镜头边界，就无法计算镜头长度、剪辑节奏、场景结构或镜头组接模式。因此它是很多自动拉片系统的第一步。

AI 任务：给定视频，模型需要找出所有切点和转场位置，并尽量区分硬切与渐变转场。

AVEMovieNet

camera motion / 镜头运动

常见类型：pan、tilt、zoom、dolly、tracking、handheld

定义：镜头运动描述摄影机或镜头焦距在拍摄过程中的变化，例如摇、移、推、拉、跟拍和变焦。

为什么重要：镜头运动不仅是视觉效果，也常承担叙事功能。例如推镜可能制造压迫或强调，跟拍可能增强沉浸感，手持镜头可能制造纪实感或不稳定感。

AI 难点：模型需要区分“物体在动”和“摄影机在动”，还要区分几何运动和语义运动意图。

CameraBenchMovieShots

optical flow / 光流

计算机视觉术语

定义：光流描述视频中像素或视觉特征在连续帧之间的运动方向和速度。

为什么重要：光流可用于估计运动强度、识别镜头运动、动作识别和相机运动分析。它是从画面运动进入视听语言分析的技术桥梁。

局限：光流主要描述几何变化，不等于理解镜头的叙事功能。它能告诉你“画面怎么动”，但不能直接告诉你“为什么这样动”。

shot-reverse-shot / 正反打

常见于对话场景

定义：正反打是一种常见剪辑模式，通常在对话中交替展示两个角色的镜头，让观众理解人物之间的空间关系和交流关系。

为什么重要：它是连续性剪辑的重要组成部分，也是人物关系、权力关系和情绪反应的基本表达方式。

AI 研究意义：识别正反打需要理解人物位置、视线方向、镜头顺序和场景连续性，难度高于单镜头分类。

Film Editing PatternsAVE

continuity editing / 连续性剪辑

相关概念：轴线关系、视线匹配、动作匹配、空间连续性

定义：连续性剪辑是一套让观众感觉空间、时间和动作连续的剪辑规则。

为什么重要：它让观众不被剪辑打断，能够自然理解人物位置、动作方向和叙事进展。

AI 研究意义：如果要评价 AI 生成视频是否“像电影”，连续性就是关键标准之一。生成视频常见问题包括人物位置跳变、视线不一致和空间关系混乱。

Film Editing PatternsCineTechBench

color histogram / 色彩直方图

传统视觉特征

定义：色彩直方图统计图像或视频帧中不同颜色范围的分布。

为什么重要：它可用于分析影片整体色调、场景变化、导演风格和类型风格。例如黑色电影、青春片、恐怖片可能有不同的色彩分布。

局限：色彩直方图不能直接解释色彩的象征意义，仍需结合场景、叙事和文化语境。

editing rhythm / 剪辑节奏

相关指标：shot length、ASL、cut frequency

定义：剪辑节奏指镜头切换的速度、密度和变化方式。它既包括平均剪辑速度，也包括节奏起伏。

为什么重要：剪辑节奏影响紧张感、情绪推进、动作强度和叙事速度。它是连接形式统计和观众感受的重要概念。

怎么分析：可结合镜头长度分布、ASL、段落内镜头密度、声音节拍和情绪曲线分析。

multimodal learning / 多模态学习

在影视研究中常见模态：video、audio、text

定义：多模态学习是把不同类型的信息共同建模，例如画面、声音、字幕、对白、剧情文本和元数据。

为什么重要：视听语言本来就是多模态的。只看画面会遗漏音乐、对白、音效和字幕在意义生成中的作用。

影视应用：片段检索、情绪识别、剧情理解、人物关系分析、预告片片段选择和媒体推荐。

MovieNetMediaFM

audio-video-text / 声画文三模态

也可理解为：声音、画面、文字共同建模

定义：把音频、视频画面和文本信息放在同一模型或同一分析框架中。文本可以来自字幕、对白转写、音频描述、剧情简介或标签。

为什么重要：影视叙事常常由画面、声音和文字共同完成。例如画面表现人物沉默，音乐制造紧张，字幕/对白给出信息，这些都需要联合理解。

MediaFMMovieNet

VLM / Vision-Language Model / 视觉语言模型

相关：MLLM、Video-Language Model、Vision-Language Understanding

定义：视觉语言模型能够同时处理视觉输入和文本输入，完成描述、问答、检索、分类和推理等任务。视频场景下也常扩展为 video-language model。

为什么重要：VLM 使模型可以回答“这个镜头是什么景别”“摄影机是否在推进”“这个片段的情绪是什么”等问题。

局限：通用 VLM 不一定懂电影语言。ShotBench、CameraBench 等 benchmark 正是为了测试这种能力边界。

ShotBenchCameraBench

video QA / 视频问答

QA = Question Answering

定义：给定一段视频和一个问题，模型需要根据视频内容回答。电影语言研究中的视频问答可以询问景别、机位、运动、构图、人物关系和叙事功能。

为什么重要：它把模型能力从单纯分类扩展到可解释的问答形式，适合评估模型是否理解细粒度视听线索。

ShotBenchCameraBench

cinematic language / 电影语言

中文语境中常与“视听语言”“镜头语言”交叉使用

定义：电影通过镜头、构图、运动、剪辑、声音、表演、色彩和叙事组织来表达意义的方式。

为什么重要：AI 研究如果只识别画面中有什么物体，还没有进入电影语言层面。电影语言强调形式如何产生叙事、情感和审美效果。

ShotBenchCineTechBench

benchmark / 基准评测

相关：dataset、leaderboard、evaluation protocol

定义：benchmark 是用于统一评估模型能力的数据集、任务和评价指标组合。

为什么重要：没有 benchmark，不同模型很难公平比较。电影语言领域的 benchmark 可以测试模型是否理解景别、运动、剪辑、构图和生成质量。

注意：benchmark 的设计会影响研究方向。只评估分类准确率，可能忽略叙事功能、文化语境和审美解释。

CNN / Convolutional Neural Network / 卷积神经网络

定义：CNN 是一种经典视觉深度学习模型，常用于图像分类、特征提取、目标识别和早期视觉任务。

影视应用：景别识别、场景分类、人物检测、图像特征提取等。虽然现在 Transformer 和 VLM 更热门，但 CNN 仍是理解早期电影视觉分析论文的重要基础。

object detection / 目标检测

定义：识别画面中的对象类别，并给出其位置框。

影视应用：检测人物、车辆、道具、武器、空间元素等，用于场景理解、人物关系分析和视觉叙事研究。

Distant Viewing ToolkitMovieNet

Transformer

定义：一种基于注意力机制的深度学习架构，广泛用于语言、视觉、音频和多模态模型。

影视应用：长视频理解、字幕-画面对齐、片段检索、视频问答、多模态嵌入和媒体基础模型。

MediaFM

contrastive learning / 对比学习

定义：让模型学习把相关的样本拉近、不相关的样本推远。例如把一个视频片段和正确文本描述拉近，把错误描述推远。

影视应用：视频-文本检索、片段匹配、预告片素材检索和多模态表示学习。

embeddings / 嵌入表示

定义：把图像、视频、声音或文本转化为向量表示，使模型可以计算相似度、聚类或检索。

影视应用：相似镜头检索、情绪片段查找、风格聚类、推荐系统和多模态媒体理解。
